# Vue Calender

# About
Vue Calender is a minimal functional calender created using vue js 3, html and tailwind csss.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
